Hide Faces

Another Cascade Machine Learning Model. This One Hides Faces Instead of Showing Them and it is Probably even More Used Than the Previous one.

This One is Very Similar to The One in Face Detection but It's Implemented in Skimage which Is a Native Python Library for Image Processing unlike OpenCV that is written in C++ and supports Java, Python, Matlab and C.



The Algorithm is The Same with One Exception:

Here When we Locate the Faces We don't draw Rectangles Around Them We Use Gaussian Filter on the results and this way Rewrite the entire Image. Keep That In Mind - Gaussian Filter is Another Ml Algorithm that Needs to Process the Data. This Will Make It Significantly slower Than the Face Detection.
